Скрипт DiffFilter служит для поиска в файлах "разниц" изменения на слова, указанные при запуске скрипта.

Параметры запуска:

DiffFilter.py file_name.rtf new_word old_word,

где DiffFilter.py - сам скрипт, file_name.rtf - имя файла "разницы", new_word - слово, на которое должно быть изменение в New, old_word - слово, на которое должно быть изменение в Old.

Пример запуска:

DiffFilter.py diff_00001.rtf just simply

С такими параметрами из файла diff_00001.rtf будут вытащены все разницы, где в новом переводе just, а в старом simply. Найденные разницы сохраняются в файл формата rtf c именем diff_new_word_old_word.rtf рядом со скриптом.

Разницы из примера будут сохранены в файле diff_just_simply.rtf

Внимание! Скрипт не будет работать с размеченными разницами и разницами, сохраненными через MSWORD. Если в разнице перед запуском скрипта нужно что-то поменять, её можно открыть в WordPad'е, таким образом не собъется разметка.
